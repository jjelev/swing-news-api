Swing News API
==============

This is testing project. Use on your own risk!  
To run the code use you may use Docker.  
```bash
docker-compose up -d
```
or just setup the environment yourself.  
### Manage the composer dependencies.
```bash
composer install
```
### Create .env file to store configuration
```bash
cp .env.example .env
```
### Run the Tests
```
make test
```
